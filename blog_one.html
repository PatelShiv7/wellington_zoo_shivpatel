Wellington Zoo Website

For my first website, I wanted to recreate the Wellington Zoo webpage. This used images from their images folder which linked to external websites on their page.

At the top of the page, it has an interactive javascript slideshow where the user can rotate around three images relating to the zoo. I've also added a navigation bar which has a:hover features in the CSS.

For the rest of the website, I've added images which mostly have a-links attached to them which link to external websites (most of these images have been taken from the Wellington Zoo website under their images folder in source control).

I've also implemented a search bar which allows the users to enter their emails for newsletter updated by using a form.
